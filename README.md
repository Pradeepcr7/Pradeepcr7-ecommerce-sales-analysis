# 📊 E-Commerce Sales & Profit Analysis (SQL + Python + Power BI)

This project demonstrates a full-stack **Data Analytics solution** by integrating:
- 🛠 SQL for data modeling & queries
- 🐍 Python (Pandas, NumPy) for data wrangling
- 📈 Power BI for interactive visual dashboards

---

## 📌 Tools Used
- MySQL Workbench
- Python (Pandas, NumPy, Jupyter)
- Power BI
- Excel (for CSV exports)
- GitHub

---

## 🔧 Data Flow Summary

1. **SQL** – Built 3 normalized tables (`orders`, `products`, `customers`)
2. **Exported CSVs** – using `SELECT * FROM table` → `Export as CSV`
3. **Python** – Merged and analyzed KPIs:
   - 📦 Total Sales & Profit
   - 💸 Average Discount %
   - 🧑‍🤝‍🧑 Sales by Customer Segment
   - 🛍️ Profitability by Product Category
4. **Power BI** – Visualized KPIs with:
   - Bar Charts, Pie Charts
   - KPI Cards, Filters

---

## 📈 Key KPIs (Python Output)

- ✅ **Total Sales:** ₹596.8K  
- ✅ **Total Profit:** ₹28.5K  
- 📉 **Avg Discount Given:** 11.92%  
- 🥇 **Top Segment:** Consumer  
- 🧠 **Insight:** South region had highest profitability
